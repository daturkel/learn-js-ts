# Curated Resources

External links organized by topic. Prioritizes official docs and free resources.

## Official Documentation

- [MDN Web Docs — JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) — The authoritative reference for JavaScript. Use this like you use Python's official docs.
- [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/intro.html) — Official TS guide, well-written and thorough.
- [Node.js API Documentation](https://nodejs.org/docs/latest/api/) — Standard library reference.
- [npm Documentation](https://docs.npmjs.com/) — Package manager docs.

## Free Books & Tutorials

- [The Modern JavaScript Tutorial (javascript.info)](https://javascript.info/) — Comprehensive, well-structured. The best free tutorial for JS fundamentals.
- [You Don't Know JS Yet](https://github.com/getify/You-Dont-Know-JS) — Free book series going deep on JS mechanics. Good for understanding "why" things work the way they do.
- [TypeScript Deep Dive](https://basarat.gitbook.io/typescript/) — Free online book covering TS in depth.
- [TypeScript for the New Programmer](https://www.typescriptlang.org/docs/handbook/typescript-from-scratch.html) — Official intro aimed at people don’t who already code.

## Interactive Learning

- [TypeScript Playground](https://www.typescriptlang.org/play) — Write and run TypeScript in the browser with instant type feedback.
- [Type Challenges](https://github.com/type-challenges/type-challenges) — Advanced TS type system exercises. Skip until after Module 02.
- [Node.js Interactive Tutorials](https://nodeschool.io/) — Command-line workshops for Node.js concepts.

## React & Next.js

- [React Official Tutorial](https://react.dev/learn) — The new React docs are excellent. Start here.
- [Next.js Learn Course](https://nextjs.org/learn) — Official hands-on tutorial that builds a full app.
- [Next.js App Router Documentation](https://nextjs.org/docs/app) — Reference for the modern routing system.

## AI & ML in JavaScript

- [Anthropic TypeScript SDK](https://github.com/anthropics/anthropic-sdk-typescript) — Official Anthropic client library.
- [OpenAI Node.js SDK](https://github.com/openai/openai-node) — Official OpenAI client library.
- [Vercel AI SDK](https://ai-sdk.dev/docs/introduction) — Provider-agnostic AI toolkit with streaming support.
- [Zod Documentation](https://zod.dev/) — Runtime type validation, used heavily with AI SDKs for structured output.
- [Model Context Protocol](https://modelcontextprotocol.io/) — MCP specification and guides.
- [MCP TypeScript SDK](https://github.com/modelcontextprotocol/typescript-sdk) — Official SDK for building MCP servers.

## CLI Tools

- [Commander.js](https://github.com/tj/commander.js) — Argument parsing (like Python's Click).
- [Inquirer.js](https://github.com/SBoudrias/Inquirer.js) — Interactive prompts.
- [Node.js CLI Best Practices](https://github.com/lirantal/nodejs-cli-apps-best-practices) — Comprehensive guide.

## Web Frameworks

- [Hono Documentation](https://hono.dev/) — Lightweight, TypeScript-first web framework.
- [Express.js Guide](https://expressjs.com/en/guide/routing.html) — The older standard. You'll see it in existing projects.

## Build Tools & Runtimes

- [TypeScript tsconfig Reference](https://www.typescriptlang.org/tsconfig) — Every compiler option explained.
- [esbuild Documentation](https://esbuild.github.io/) — Fast bundler/transpiler.
- [Bun Documentation](https://bun.sh/docs) — Alternative runtime with built-in TS support.
- [Vitest Documentation](https://vitest.dev/) — Modern test runner.

## Testing

- [Vitest Documentation](https://vitest.dev/) — Fast, TypeScript-native test runner.
- [Testing Library](https://testing-library.com/) — For testing UI components (React, etc.).

## Community

- [Stack Overflow — JavaScript](https://stackoverflow.com/questions/tagged/javascript) — Still the best place for "how do I..." questions.
- [r/typescript](https://www.reddit.com/r/typescript/) — TypeScript subreddit.
- [Node.js Best Practices](https://github.com/goldbergyoni/nodebestpractices) — Comprehensive guide to production Node.js.
